Docker的安装

在linux虚拟机上安装docker

  1. 检查内核版本，必须是3.10及以上uname -r

  2. 安装dockeryum install docker

  3. 输入y确认安装

  4. 启动dockersystemctl start docker

  5. 查看docker版本docker -v

  6. 默认安装在/var/lib/docker

Docker的卸载

  1. 首先搜索已经安装的docker 安装包yum list installed|grep docker或者rpm -qa|grep docker

  2. 分别删除安装包 sudo yum –y remove 安装包

  3. 删除镜像/容器等 rm -rf /var/lib/docker
